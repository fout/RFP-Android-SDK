CHANGELOG

v3.8.0 (2020-10-16)

  • アウトストリーム動画広告配信機能を追加

v3.7.0 (2020-03-23)

  • ExoPlayer 2.10 および 2.11 へ対応
  • ExoPlayer 2.4 および 2.5 のサポートを終了

v3.6.1 (2019-09-12)

  • イベント送信時にクラッシュする場合がある問題を修正

v3.6.0 (2019-08-27)

  • 動画広告がデタッチされた時の再生状態を通知するVideoAdView#OnDetachedListenerを追加。

  • 動画広告をデタッチされた際の状態から再生するメソッドVideoAdView#processAd(RFPInstreamInfoModel adModel, @Nullable Bundle savedState)を追加。

v3.5.0 (2019-08-19)

  • 広告のロードに対するインターフェースRFPInstreamAdLoaderを追加。
  • ExoPlayer 2.9.xをサポート対象に追加。
  • 細かい不具合の修正

v3.4.4 (2019-01-16)

  • 広告取得用AsyncTaskのリーク問題を修正
  • デモアプリの公開
  • ExoPlayer 2.8.xをサポート対象に追加
  • 細かい不具合の修正

v3.4.3 (2018-11-26)

  • loadAd時のリーク問題を修正
  • 細かい不具合の修正

v3.4.2 (2018-10-29)

  • Android Lollipopの一部端末にて初期化時にクラッシュする問題を修正

v3.4.1 (2018-10-02)

  • 起動処理を改善
  • インプレッション送信時にクラッシュする場合がある問題を修正

v3.4.0 (2018-08-27)

  • 外部アドネットワーク対応

v3.3.2 (2018-08-15)

  • アイコン画像に関連する機能を削除

v3.3.1 (2018-07-11)

  • クリエイティブサイズ取得メソッドを追加
    • RFPInstreamInfoModel#creative_width()
    • RFPInstreamInfoModel#creative_height()

v3.3.0 (2018-04-26)

Changes & Enhancements

  • HTTP requestのタイムアウト時間設定を追加
  • ExoPlayer 2.7.xをサポート対象に追加。

v3.2.2 (2018-04-05)

  • 動画広告で不正なヘッダを検出した場合にリクエストを再送するように修正
  • 動画プレイヤー内部エラーをVideoAdView#OnErrorListenerで検知できるように修正
  • VideoAdViewのレイアウトエディタ上での表示に関する問題を修正

v3.2.1 (2018-01-26)

  • ExoPlayer 2.6.xをサポート対象に追加。

v3.2.0 (2017-11-20)

  • 動画広告のキャッシュ制御を追加

v3.1.0 (2017-10-30)

  • 広告ID取得に対応

v3.0.1 (2017-10-24)

  • 安定性の向上

v3.0.0 (2017-10-12)

  • 動画広告SDKを分離

v2.2.2 (2017-09-07)

  • 一部の端末でRFP.init()が失敗する問題を修正。

v2.2.1 (2017-09-05)

  • ExoPlayer 2.5.xをサポート対象に追加。
  • 動画広告読み込みエラーに対するインターフェースを追加。
  • 細かい不具合の修正。

v2.2.0 (2017-08-28)

  • アクションボタンテキストの可変に対応。
  • HTTP Live Streaming形式の動画広告に対応。

v2.1.4 (2017-08-23)

  • 新しい動画広告設定時に古い動画広告が表示されないように修正。

v2.1.3 (2017-08-22)

  • ランディングページ移動時、全画面遷移時にバックグラウンドで動画再生が続いている問題を修正。

v2.1.2 (2017-08-21)

  • 全画面からの復帰時の再生位置を保持してしまう問題を修正。

v2.1.1 (2017-08-18)

  • 動画広告が複数ある場合に動画が表示されなくなってしまう問題を修正。

v2.1.0 (2017-08-17)

  • minSdkVersionを16に更新。
  • 動画の再生にTextureViewを使用するように変更。
  • ExoPlayerへの依存が追加。

v2.0.1 (2017-08-08)

  • 一部のトラッキングイベントが正しく送られていなかった問題を修正。
